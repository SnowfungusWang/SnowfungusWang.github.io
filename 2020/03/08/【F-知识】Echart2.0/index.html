SE3的统计图

tooltip

​ 可以自定义位置、内容，传入http元素

  • trigger：不同图的类型可以是item或者axis，选择合适的属性值，可以使传入值变成图标对应的部分。

  • formatter：用来设定格式，可以是格式化字符串（不同的字符代表不同的数据部分）或者http元素。

    • 函数传入的内容是这部分的参数，如果你不加限制的话鼠标移动就会导致新的事件
    • http元素的字符串不要乱打tab或者enter，会反映在tooltip。

  • extraCssText:'width:25em; white-space:pre-wrap',换行和控制宽度。

  • confine：好用，是否将 tooltip 框限制在图表的区域内。

    • 当图表外层的 dom 被设置为 'overflow: hidden'，或者移动端窄屏，导致 tooltip 超出外界被截断时，此配置比较有用。

  • position：

    • Array

      通过数组表示提示框浮层的位置，支持数字设置绝对位置，百分比设置相对位置。

      示例:

    • Function

      回调函数

      (point: Array, params: Object|Array.<Object>, dom: HTMLDomElement, rect: Object, size: Object) => Array

      可以实现和鼠标位置或者图标元素有关的位移。

渐变色

前4个参数是位置，当前为从左到右，不同的offset位置，代表该百分数位置的颜色。

Event

可以有各种鼠标事件。
